# cc3000-wifi-driver

Early development. Vendored from Adafruit_CC3000_Library; rewrite in progress.

## Supported boards

| Board | Architecture | FQBN |
|-------|--------------|------|
| Arduino UNO R3 | AVR | `arduino:avr:uno` |
| Arduino UNO R4 Minima | Renesas RA4M1 | `arduino:renesas_uno:minima` |

## When NOT to use this

If you have a working sketch on Adafruit_CC3000_Library, stay there. This library is under active development and may introduce breaking changes. The original Adafruit library is stable and widely tested; migrate only when this library reaches a stable release.
